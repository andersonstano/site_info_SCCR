O que foi utilizado nesse site?

Conheça as ferramentas utilizadas para criar esse site:

WebMatrix 3

Ferramenta disponibilizada pela Microsoft para auxiliar na criação, publicação e manutenção de sites. Conta com um sistema de módulos, componentes e templates que facilitam o gerenciamento do site como um todo.

Oferece suporte a ASP.NET, PHP, Node.js e HTML 5. Oferece também integração com o IIS Express facilitando a implantação e testes locais.

É uma ferramenta gratuita.

Bootstrap

É uma coleção de ferramentas para criação de sites e aplicações WEB. Utilizando HTML, CSS e JavaScript oferece diversos templates para tipografia, formulários, botões, menus e componentes de interface. Oferece layouts flexíveis que se adaptam a diversos dispositivos e diversos tamanhos de tela.

Criado originalmente por um designer e por um desenvolvedor do Twitter - @mdo e @fat - se tornou um dos mais famosos framework e projeto open-source do mundo.

É uma ferramenta gratuita.

Heroku

É um serviço de computação em nuvem que disponibiliza plataformas para hospedagem de sites e aplicações WEB.

Tem suporte a Java, Node.js, Django, Ruby, Facebook, dentre outras e oferece add-ons como banco de dados, monitoramento e utilitários.

É uma ferramenta gratuita, desde que respeitados alguns limites.

O código fonte desse site está disponível no GitHub

É possível hospedar sites com conteúdo estático no Heroku. ;)
