SCCR - Simulação de consumo e carga para residências

Entenda a motivação para criação deste serviço e o que foi utilizado para criá-lo.

Para que utilizar o SCCR

A utilidade de uma simulação de consumo residencial não precisa de muita explicação. Com a estimativa de consumo é possível:

Estimar o gasto mensal;

Identificar quais equipamentos consomem mais;

Verificar quanto um novo equipamento irá consumir;

Simular novos hábitos de consumo.

Uma das principais dúvidas dos consumidores de energia elétrica ao solicitarem serviços como Ligação e Ligação Direta é informar a carga instalada nas residências. Essa informação é utilizada para determinar o faseamento da residência, definir padrões de vistoria e para cobrança de alguns serviços. Caso a carga seja informada erroneamente, o consumidor pode pagar mais pelos serviços ou pode ter problemas em suas instalações elétricas, como desarme de disjuntores e subtensão.

O serviço SCCR exporta, a partir de um banco MySQL, uma lista de equipamentos contendo a desrição do equipamento, potência, fator de potência e a quantidade de dias e horas de uso.

Com o web service SCCR fica fácil simular a carga residencial. E com essa informação evita-se os problemas apontados.

Tecnologias utilizadas

Para desenvolvimento do web service utilizamos o Netbeans IDE , ferramenta que auxilia o desenvolvimento, agregando ferramentas e facilitadores. Ele é um projeto gratuito e open-source.

O código está disponibilizado publicamente no GitHub , ferramenta online para colaboração, revisão e gerenciamento de código que inclui planos gratuitos e pagos. É possivel integrar o repositório do GitHub no Netbeans - veja como
